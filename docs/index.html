Typography ucsb.edu/visual-identity/typography

This code references the offical UCSB web fonts by name, Avenir and Produkt, but does not include the font files. You must set up the font files yourself which is outside the scope of this project.

The style will not be applied just by including the css file. You must explicitly use the css classes.

You may not be able to use the offical font so this code will cascade to similar Google fonts, Nunito Sans and Sanchez, and then to Century Gothic and sans-serif.
